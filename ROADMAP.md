# Roadmap

## Phase 1
- Multi-user support with crews and foreman role
- Photo download to S3-compatible storage (replace file_id storage)
- Webhook deployment (nginx + systemd)
- Auto clock-out / scheduled reminders
- Admin panel (FastAPI)

## Phase 2
- Web dashboard with shift calendar
- Voice transcription (Whisper) for notes
- Multi-language UI (pl, uk)
- Push notifications beyond bot replies
- Telegram inline mode

## Phase 3
- Billing / subscriptions
- Mobile app companion
- Integrations (1C, accounting systems)
